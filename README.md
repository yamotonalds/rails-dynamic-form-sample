## Railsで1対多のモデルに対するフォームで動的に入力項目を増やす実装のサンプル

各branchが実装サンプルになっている。

- static-rails
  素のRailsでの非動的な実装。これをbaseに他のbranchで動的にする実装をしている。
- dynamic-jquery
  jQueryでの実装。
- dynamic-cocoon
  cocoon gemでの実装。
- dynamic-vue
  Vue.jsを局所的に使用した実装。
